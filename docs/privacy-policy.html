Privacy Policy

Last updated: March 27, 2026

Am I Spoiler? ("the Extension") is a Chrome extension that analyzes YouTube video thumbnails to detect potential spoilers using AI. This Privacy Policy explains what data the Extension accesses and how it is used.

Data We Access

When you click the "Spoiler?" button on a YouTube video thumbnail, the Extension reads the YouTube video ID from the current page URL and sends it to our backend server (am-i-spoiler.vercel.app) for analysis.

This is the only data transmitted. We do not collect:

  • Personal identification information (name, email, address)
  • Browsing history or watch history
  • Authentication credentials
  • Financial or payment information
  • Any data unrelated to the video being analyzed

Data Storage

The Extension does not store any user data permanently. Video IDs sent to the backend are used solely to perform the spoiler analysis and are not logged, saved, or shared with third parties.

Third-Party Services

To perform analysis, the backend server communicates with the following services:

  • YouTube Data API — to fetch video metadata
  • Anthropic Claude API — to perform AI-based spoiler analysis
  • Serper API — to search for airing status information
  • TMDB API — to retrieve streaming platform availability

All API keys for these services are stored securely on the server and are never exposed to the user or transmitted through the Extension.

Permissions

  • activeTab — to read the current YouTube page URL and extract the video ID
  • scripting — to inject the Spoiler button into YouTube video thumbnails
  • storage — to temporarily hold analysis state during a session
  • youtube.com host permission — to operate on YouTube pages
  • am-i-spoiler.vercel.app host permission — to communicate with the analysis backend

Children's Privacy

This Extension is not directed at children under 13 and does not knowingly collect any personal data from minors.

Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be reflected by updating the date at the top of this page.
